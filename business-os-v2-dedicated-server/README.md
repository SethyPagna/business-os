# Atlas V2: Dedicated Server Per Business

This profile is for:
- one full server per business
- one organization currently active in that server instance
- business can migrate the whole app/runtime together

Use URL pattern:
- Admin: `https://atlas.tail47edcd.ts.net/`
- Public: `https://atlas.tail47edcd.ts.net/public` or configured custom public path

This is the closest match to the current Business OS deployment model.
